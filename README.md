# ✈️ Trillo: Flexbox Masterclass

A sleek, modern front-end interface for a fictional travel booking application. This project was developed with a singular, intense focus: to achieve absolute mastery over CSS Flexbox, utilizing it exclusively to build every layout and component within the application.

*Note: This project was built by following the "Advanced CSS and Sass" course by Jonas Schmedtmann. I manually implemented the entire architecture, writing every line of code to deeply understand and internalize advanced layout behaviors.*

## ✨ Technical Highlights & Flexbox Focus
* **Total Flexbox Architecture:** Every UI component—from the global page layout down to the smallest navigation items and user reviews—is structurally built using Flexbox.
* **Property Mastery:** Extensive and deliberate use of all Flexbox properties (`flex-grow`, `flex-shrink`, `flex-basis`, `align-items`, `justify-content`, `order`, etc.) to control precise element behavior without relying on traditional floats or external grids.
* **Advanced SASS Integration:** Engineered using SASS (SCSS) following the highly maintainable **7-1 Architecture pattern**.
* **BEM Methodology:** Strict adherence to the Block Element Modifier (BEM) naming convention, ensuring clean, scalable, and isolated CSS code.
* **Fluid Layouts:** A seamless, responsive design that adapts organically to different screen sizes without heavy reliance on media queries.

## 🛠️ Technologies Used
* HTML5
* CSS3
* Advanced Flexbox
* SASS / SCSS (7-1 Architecture)
* BEM Naming Convention

## 🚀 Learning Outcome
While the design was provided as part of the guided curriculum, the implementation was my training ground for advanced layout mechanics. Building Trillo gave me the confidence to construct complex, multi-layered interfaces using pure CSS and Flexbox, proving my capability to write clean, scalable, and highly organized UI code suitable for large-scale projects.
